# 🔍 雨夜密室杀人案 - 侦探推理游戏

一个基于浏览器的文字冒险侦探推理游戏。玩家扮演警探林正，在暴雨之夜调查一起密室杀人案，通过收集线索、盘问嫌疑人、分析证据，最终揭开真相。

## 🎮 在线游玩

**[点击这里开始游戏](https://你的用户名.github.io/你的仓库名/)**

## 📖 游戏特色

- **32个剧情场景** — 丰富的分支剧情，每次选择都会导向不同的调查路径
- **5种结局** — 完美破案 / 部分成功 / 错判无辜 / 真凶逃脱 / 意外反转
- **5名嫌疑人** — 管家老周、陈夫人、赵明辉、刘师傅、神秘访客，每人都有隐藏的秘密
- **道具系统** — 收集关键证据解锁隐藏选项
- **HP生命值系统** — 危险选择会消耗生命值，归零则调查失败
- **随机事件** — 部分场景触发随机事件，增加重玩性
- **线索值系统** — 积累线索解锁隐藏选项和嫌疑人秘密
- **指认环节** — 支持多选指认，组合不同嫌疑人触发不同结局

## 🎯 游戏玩法

1. 阅读场景描述，了解当前情况
2. 从可用选项中选择行动方向
3. 收集道具、积累线索值
4. 访问关键调查场景解锁更多选项
5. 在指认环节选择你认为有罪的嫌疑人
6. 根据收集的证据和指认结果获得不同结局

**提示：** 按数字键 1-5 快速选择，按 S 键查看嫌疑人档案。

## 🚀 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
```

2. 使用任意 HTTP 服务器运行（因为需要加载 JS 模块）：
```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server
```

3. 打开浏览器访问 `http://localhost:8000`

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript，无需任何框架
- 响应式设计，支持桌面和移动端
- 使用 CSS 变量和动画实现现代 UI
- 基于原 C 语言版本完整移植
